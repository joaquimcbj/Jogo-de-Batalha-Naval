Super Trunfo - Países e Cidades
Este projeto em Linguagem C é um simulador do jogo Super Trunfo, desenvolvido para o desafio de lógica e manipulação de dados.

Funcionalidades
Cadastro de Cartas: Entrada de dados detalhados de cidades.

Cálculos Automáticos: Implementação de fórmulas matemáticas para Densidade Populacional e PIB per Capita.

Sistema de Comparação: Lógica para comparar atributos entre duas cartas e determinar a vencedora.
